{\footnotesize\ttfamily void integrate (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [virtual]}}



Propagate system for a time step. 

This function integrates equations of motion as introduced in Vicsek model see, e.\+g., Eqs. (13) and (14) in T. Vicsek, A. Zafeiris, Phys. Reports 517 (2012) p. 71-\/140. Actual equations are a bit modified in order to introduce spacial interaction term $ \vec F_i = \sum_{i\neq j}\vec F_{ij} $. $ \vec v_i(t+1) = v_0\frac{\langle\vec v_j(t)\rangle}{\left|\langle\vec v_j(t)\rangle\right|} + \mu \sum_{i\neq j}\vec F_{ij} + \mathrm{noise} $ and $ \vec r_i(t+1) = \vec r_i(t) + \vec v_i(t+1) $. $ \langle\vec v_j(t)\rangle $ is the average velocity of the neighbouring particles in some radius and $ v_0 $ is magnitude of the velocity. $ \mu $ is the mobility. We note that Viscek model has no explicit time step, i.\+e. it is set to one. Noise term is essentially a random rotation of the particle\textquotesingle{}s velocity around its normal vector by angle $ \delta\vartheta $ where $ \delta\vartheta $ is a random variable drawn uniformly from the interval $ [-\eta\pi\dots\eta\pi] $, where $ \eta $ is a constant.
