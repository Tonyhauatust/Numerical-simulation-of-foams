\doxysubsection{Detailed Description}
This is a parser for parsing command that contain pairwise alignment. Structurally, it is very similar to the command parsed (\begin{DoxySeeAlso}{See also}
\doxyref{parse\+\_\+command.\+hpp}{p.}{parse__command_8hpp}) This parser extracts the pairwise aligner line type and returns it together with the list of parameters that will be parsed separately.
\end{DoxySeeAlso}
for example\+:

pair\+\_\+align mf \{ J = 1.\+0; \}

This parser will extract the pairwise aligner type (in this case \char`\"{}mf\char`\"{}) and return the rest of the line for post-\/processing.
