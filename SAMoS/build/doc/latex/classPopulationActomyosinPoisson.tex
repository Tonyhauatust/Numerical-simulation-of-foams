This funciton controls attachment (note\+: function name derives from the intial intent of the population classes to use to treat cell division) 

This function controls attachement and detachement of myosin beads. This is a achieved via chaning types of myosin head groups from attached (type \char`\"{}\+A\char`\"{}) to detached (type \char`\"{}\+D\char`\"{})a and vice versa. In this implementation we use a Poisson process.

\begin{DoxyNote}{Note}
In order to avoid head groups, e.\+g. chainging their type from \char`\"{}\+A\char`\"{} to \char`\"{}\+D\char`\"{} and back to \char`\"{}\+A\char`\"{} in the single time step, we implement both processes in the same function using lists of indices. 
\end{DoxyNote}
