\doxysubsection{Detailed Description}
This is a parser for parsing command that contain simulation box information Structurally, it is very similar to the command parsed (\begin{DoxySeeAlso}{See also}
\doxyref{parse\+\_\+command.\+hpp}{p.}{parse__command_8hpp}) This parser extracts the potential line type and returns it together with the list of parameters that will be parsed separately.
\end{DoxySeeAlso}
for example\+:

box periodic \{ lx = 10.\+0; ly = 10.\+0; lz = 1.\+0; \}

This parser will extract the box type (in this case \char`\"{}periodic\char`\"{}) and return the rest of the line for post-\/processing.
