\doxysubsection{Detailed Description}
This is a parser for parsing command that contain population control. Structurally, it is very similar to the command parsed (\begin{DoxySeeAlso}{See also}
\doxyref{parse\+\_\+command.\+hpp}{p.}{parse__command_8hpp}) This parser extracts the \doxyref{population}{p.}{classpopulation__grammar_af05a9240f124a0fa5ead38c41c91001e} line type and returns it together with the list of parameters that will be parsed separately.
\end{DoxySeeAlso}
for example\+:

population random \{ group=all; division\+\_\+rate=2.\+0; death\+\_\+rate = 0.\+5; freq = 100 \}

This parser will extract the population type (in this case \char`\"{}random\char`\"{}) and return the rest of the line for post-\/processing.
