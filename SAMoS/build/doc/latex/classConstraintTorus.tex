\doxysubsection{Detailed Description}
Enforces all particles to be on the surface of a torus of radius $ c $ centred at the origin with the tube radius $ a $. All velocities will point in the tangent direction.

Computes normal to the surface. 

Compute normal to the surface at p

Normal vector is computed as the normalized gradient vector at point p. Coordinates of gradient vector are given as\+:

$ g_x = -\frac{2x\left(c-\sqrt{x^2+y^2}\right)}{\sqrt{x^2+y^2}}, $ $ g_y = -\frac{2y\left(c-\sqrt{x^2+y^2}\right)}{\sqrt{x^2+y^2}}, $ $ g_z = 2z. $ ~\newline
