\doxysubsection{Detailed Description}
\doxyref{Bond\+Fene\+Potential}{p.}{classBondFenePotential} implements standard F\+E\+ME bond potential \doxyref{Potential}{p.}{classPotential} is given as $ U_{fene}\left(r_{ij}\right) = -\frac{1}{2}kr_0^2\log\left[1-\left(\frac{\left|\vec r_{ij}\right|}{r_0}\right)^2\right] + WCA(r_{ij})$, where $ k $ is the spring constant, $ r_0 $ is the the maximum bond length and $ \vec r_{ij} = \vec r_i - \vec r_j $. W\+CA is the repulsive part of Lennard-\/\+Jones potential (so called, Weeks-\/\+Chandler-\/\+Andersen potential), which is {\itshape N\+OT} included here. Therfore, the user has to make sure to disable exclusions in bonds in order to properly treat the short-\/range part of the potential.
