\doxysubsection{Detailed Description}
\doxyref{Pair\+Rod\+Potential}{p.}{classPairRodPotential} implements the anisotropic soft repulsion between rods of length $ l $ and radius $ \sigma $. Details of the interaction potential, force and torques are straightforward to derive, but lengthy. They are given in a separate set of notes in the docs directory.

\begin{DoxyNote}{Note}
Rod length is encoded within the potential and not as a part of the \doxyref{Particle}{p.}{classParticle} class as for the time being we would like to keep the core of the code work with spherical objects only. If the anisotropy turns out to be important, this may change and the \doxyref{Particle}{p.}{classParticle} class might start carrying quaternion describing its orientation. This would allow of a uniform interface for implementing arbitrary anisotropic potentials.
\end{DoxyNote}
