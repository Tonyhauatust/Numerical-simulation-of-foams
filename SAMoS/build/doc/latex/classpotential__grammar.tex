\doxysubsection{Detailed Description}
This is a parser for parsing command that contain pair potentials. Structurally, it is very similar to the command parsed (\begin{DoxySeeAlso}{See also}
\doxyref{parse\+\_\+command.\+hpp}{p.}{parse__command_8hpp}) This parser extracts the \doxyref{potential}{p.}{classpotential__grammar_adf2ddbd7ed8feb1a838f831b0f96bd63} line type and returns it together with the list of parameters that will be parsed separately.
\end{DoxySeeAlso}
for example\+:

pair\+\_\+potential lj \{ eps = 1.\+0; sigma = 1.\+0; r\+\_\+cut = 2.\+5; \}

This parser will extract the potential type (in this case \char`\"{}lj\char`\"{}) and return the rest of the line for post-\/processing.
