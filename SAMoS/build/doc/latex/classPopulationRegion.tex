\doxysubsection{Detailed Description}
\doxyref{Population\+Region}{p.}{classPopulationRegion} class handles removal of cells that are outside a given region of space. If coordinates of a cell fall outside the \char`\"{}allowed\char`\"{} region, the cell is marked for removal a removed from the system. Boundaries are updated accordingly.
