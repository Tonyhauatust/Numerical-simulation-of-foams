\doxysubsection{Detailed Description}
Enforces all particles to be on the surface of a peanut shaped oval. It is generated by rotating a Cassini oval curve around x axis. The surface is given in the implicit for as $ f(x,y,z) = (x^2+y^2+z^2)^2 - 2a^2(x^2-y^2-z^2)+a^4-b^4 $, where $ a $ and $ b $ control shape of the oval. Note that $ a < b $ in order for the oval not to be disconnected.
