\doxysubsection{Detailed Description}
\doxyref{Population\+Actomyosin}{p.}{classPopulationActomyosin} class handles a basic model for actomyosin binding and unbiding. This is inteded to be a crude approximation of the phosphorealtion processes that lead to different binding affinities for myosin motors.

This function controls attachement and detachement of myosin beads. This is a achieved via chaning types of myosin head groups from attached (type \char`\"{}\+A\char`\"{}) to detached (type \char`\"{}\+D\char`\"{})a and vice versa.

\begin{DoxyNote}{Note}
In order to avoid head groups, e.\+g. chainging their type from \char`\"{}\+A\char`\"{} to \char`\"{}\+D\char`\"{} and back to \char`\"{}\+A\char`\"{} in the single time step, we implement both processes in the same function using lists of indices.
