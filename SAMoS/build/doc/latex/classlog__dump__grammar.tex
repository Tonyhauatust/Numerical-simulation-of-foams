\doxysubsection{Detailed Description}
This is a parser for parsing command that contain log or dump directives. Both cases are parsed with the same parser as both log and dump commands have identical syntax

For example\+:

log terminal \{ freq = 100; step; temperature; potential\+\_\+energy \} dump coord \{ freq = 100; type = xyz; file = multi \}

This parser will extract log or dump file name (in this case \char`\"{}terminal\char`\"{} and \char`\"{}coord\char`\"{}) and return the rest of the line for post-\/processing.
