\doxysubsection{Detailed Description}
\doxyref{Pair\+Line\+Tension\+Potential}{p.}{classPairLineTensionPotential} implements the line tension term in the model for an active tissue model. Force only exists along boundary edges and is proportional to the length of the edge, i.\+e., $ \vec f_i = \lambda \left|\vec r_{ij}\right|\hat{\vec r}_{ij} $ and $ \vec f_j = -\lambda \left|\vec r_{ij}\right|\hat{\vec r}_{ij} $, where $ \vec r_{ij} $ if the vector along the edge connecting vertices belong to it.
