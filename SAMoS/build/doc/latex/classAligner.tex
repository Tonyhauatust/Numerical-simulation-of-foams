\doxysubsection{Detailed Description}
\doxyref{Aligner}{p.}{classAligner} class handles all director aligners (pairwise and external field type) present in the system. All aligners are stored in two S\+TL maps which both have strings as keys while items are pointers (boost shared\+\_\+ptr) to the \doxyref{Pair\+Align}{p.}{classPairAlign} and \doxyref{External\+Align}{p.}{classExternalAlign} abstract classes, respectively. The integrator part of the code calls member functions of this class to perform actual alignment computation.
