\doxysubsection{Detailed Description}
This is a parser for parsing command that contain group directives.

For example\+:

group small \{ type = 1 \} group R2 \{ radius = 2.\+0 \}

This parser will extract group name name (in this case \char`\"{}small\char`\"{} and \char`\"{}\+R2\char`\"{}) and return the rest of the line for post-\/processing.
