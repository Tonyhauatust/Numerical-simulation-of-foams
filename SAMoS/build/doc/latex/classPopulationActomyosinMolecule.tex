\doxysubsection{Detailed Description}
\doxyref{Population\+Actomyosin\+Molecule}{p.}{classPopulationActomyosinMolecule} class handles unbinding of the entire myosin molecule form the rest of the system. If any of the beads is in the attached state all head groups detach. This detachment is done with a given probability. This is to model experiments of Christoph Schmidt.
