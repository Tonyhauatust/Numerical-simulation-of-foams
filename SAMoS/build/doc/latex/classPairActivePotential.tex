\doxysubsection{Detailed Description}
\doxyref{Pair\+Active\+Potential}{p.}{classPairActivePotential} implements, what we call active potential, while this is actually not a potential. The idea behind this is to emulate contractile and extensile forces in active systems. Force is given as $ \vec F_{active} = \frac {1}{2}\alpha\left(\vec n_i-\vec n_j \right)\vec n_i \cdot \vec n_j $ if $ \vec n_i \cdot \vec n_j < 0 $, where $ \alpha $ is a parameter that determines if the force is contractile ( $ \alpha < 0 $) or extensile ( $ \alpha < 0 $). $ \vec n $ is the director vector.
