\doxysubsection{Detailed Description}
This is a parser for parsing command that contain bond potentials. Structurally, it is very similar to the command parsed (\begin{DoxySeeAlso}{See also}
\doxyref{parse\+\_\+command.\+hpp}{p.}{parse__command_8hpp}) This parser extracts the potential line type and returns it together with the list of parameters that will be parsed separately.
\end{DoxySeeAlso}
for example\+:

bond harmonic \{ k = 1.\+0; l0 = 1.\+0 \}

This parser will extract the bond potential type (in this case \char`\"{}harmonic\char`\"{}) and return the rest of the line for post-\/processing.
