\doxysubsection{Detailed Description}
\doxyref{Vertex}{p.}{structVertex} class is a light-\/weight class that handles vertices in a mesh. A vertex can simply be a particle position or the position of the dual lattice in tissue models.

This may apper as doubling of the data strucures, but gives us fexibilty to work with meshes that do not coincide with the lattice positions.
