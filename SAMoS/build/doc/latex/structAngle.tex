\doxysubsection{Detailed Description}
\doxyref{Angle}{p.}{structAngle} structure This is a basic data structure that hold information about angle in the system. This is used to model bending rigidity in chain-\/like molecules, such as polymers. \doxyref{Angle}{p.}{structAngle} holds three indices, i, j and k which are id\textquotesingle{}s of particles that are bonded. It also includes bond type, which is a positive integer.
