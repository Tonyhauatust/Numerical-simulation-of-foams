\doxysubsection{Detailed Description}
This is a parser for parsing command that contain external alignment. Structurally, it is very similar to the pair potential parser (\begin{DoxySeeAlso}{See also}
\doxyref{parse\+\_\+potential.\+hpp}{p.}{parse__potential_8hpp}) This parser extracts the external alignment line type and returns it together with the list of parameters that will be parsed separately.
\end{DoxySeeAlso}
for example\+:

external\+\_\+align gravity \{ g = 9.\+81 \}

This parser will extract the aligner type (in this case \char`\"{}gravity\char`\"{}) and return the rest of the line for post-\/processing.
