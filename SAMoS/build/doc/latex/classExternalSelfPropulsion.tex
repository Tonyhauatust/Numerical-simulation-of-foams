\doxysubsection{Detailed Description}
\doxyref{External\+Self\+Propulsion}{p.}{classExternalSelfPropulsion} handles active motion where each particle receives a force equal to $ \vec F_i = \alpha \vec n_i $, where $ \alpha $ is the activity and $ \vec n_i $ is the director (polarisation). \begin{DoxyNote}{Note}
This is not a potential, but to avoid code bloating we place it in the same class structure as regular external potential. 
\end{DoxyNote}
