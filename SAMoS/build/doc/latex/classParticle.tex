\doxysubsection{Detailed Description}
\doxyref{Particle}{p.}{classParticle} class This is a basic unit in the simulation. It contains relevant parameters that describe each active particle, such as its position, velocity, type, etc. \begin{DoxyNote}{Note}
particle coordinates are given in the external, 3d Euclidean space. Its confinement to a given manifold (curved space) is enforced via constraints. 
\end{DoxyNote}
