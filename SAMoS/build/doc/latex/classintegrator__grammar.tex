\doxysubsection{Detailed Description}
This is a parser for parsing command that contain controls the integrator. Structurally, it is very similar to the potential parser (\begin{DoxySeeAlso}{See also}
\doxyref{parse\+\_\+potential.\+hpp}{p.}{parse__potential_8hpp}) This parser extracts the \doxyref{integrator}{p.}{classintegrator__grammar_a6b378470368d1a351e53163afd6d72a2} type and returns it together with the list of parameters that will be parsed separately.
\end{DoxySeeAlso}
for example\+:

integrator nvt \{ dt = 0.\+005; Tstart = 1.\+0; Tend = 0.\+5; tau = 0.\+5; \}

This parser will integrator type (in this case \char`\"{}nvt\char`\"{}) and return the rest of the line for post-\/processing.
