\doxysubsection{Detailed Description}
This is a parser for parsing command that contain imposed constraint

For example\+:

constraint sphere \{ R = 10.\+0 \}

This parser will extract the constraint type (in this case \char`\"{}sphere\char`\"{}) and return the rest of the line for post-\/processing.
