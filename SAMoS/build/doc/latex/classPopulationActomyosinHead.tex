\doxysubsection{Detailed Description}
\doxyref{Population\+Actomyosin\+Head}{p.}{classPopulationActomyosinHead} class handles a basic model for Actomyosin head binding and unbinding. This is intended to be a crude approximation of the phosphorealtion processes that lead to different binding affinities for myosin motors. Unlike simple \doxyref{Population\+Actomyosin\+Head}{p.}{classPopulationActomyosinHead} this class only binds/unbinds at the predetermined times that are drawn from the logarithmic distribution.
