\doxysubsection{Detailed Description}
This is a parser for parsing command that contain external potentials. Structurally, it is very similar to the pair potential parser (\begin{DoxySeeAlso}{See also}
\doxyref{parse\+\_\+potential.\+hpp}{p.}{parse__potential_8hpp}) This parser extracts the potential line type and returns it together with the list of parameters that will be parsed separately.
\end{DoxySeeAlso}
for example\+:

pair\+\_\+potential gravity \{ g = 9.\+81 \}

This parser will extract the potential type (in this case \char`\"{}gravity\char`\"{}) and return the rest of the line for post-\/processing.
