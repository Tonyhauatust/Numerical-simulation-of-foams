\doxysubsection{Detailed Description}
\doxyref{Bond}{p.}{structBond} structure This is a basic data structure that hold information about bonds in the system. This is used to model chain-\/like molecules, such as polymers. \doxyref{Bond}{p.}{structBond} holds two indices, i and j which are id\textquotesingle{}s of particles that are bonded. It also includes bond type, which is a positive integer.
