\doxysubsection{Detailed Description}
\doxyref{Pair\+Soft\+Attractive\+Potential}{p.}{classPairSoftAttractivePotential} implements the pair interaction which is soft in nature with an attractive part and soft repulsive core. \doxyref{Potential}{p.}{classPotential} is given as $ U_{soft}\left(r_{ij}\right) = \frac{k}{2} \left(r_{ij} - r_e \right)^2 $, if $ r_{ij} \le r_{tunr} $, $ U_{soft} = -\frac{k}{2} \left(r_{ij} - r_f \right)^2 $ (with $ r_f = 2 r_{turn} - r_e $)if $ r_e < r_{ij} < r_f $ and zero otherwise. $ k $ is the potential strength. Often, it is convenient to set $ r_e = a_i + a_j $, with $ a_i $ and $ a_j $ being the radii of the two particles. Finally, $ r_{ij} $ is the Euclidean interparticle distance. $ r_{turn} = fr_e $, where $ f $ is a parameter given by the user.
