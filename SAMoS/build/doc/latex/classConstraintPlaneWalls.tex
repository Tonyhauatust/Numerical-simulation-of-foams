\doxysubsection{Detailed Description}
Enforces all particles to lay on the xy plane of size Lx x Ly Periodic boundary in the y-\/direction and repulsive walls at -\/L and L in the x-\/direction. z-\/coordinate set to zero.

Enforce constraint. 

Force all particles to be confined to the surface of the xy plane by setting z = 0, vz = 0, fz = 0. At the walls, particle is stopped at the wall by setting $ v_x = 0 $ and x-\/coordinate is set back to l or -\/l.
