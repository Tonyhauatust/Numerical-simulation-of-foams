\doxysubsection{Detailed Description}
Enforces all particles to be on the surface of an ellipsoid with parameters $ a $, $ b $ and $ c $ All velocities will point in the tangent direction.

Computes normal to the surface. 

Compute normal to the surface at p

Normal vector is computed as the normalized gradient vector at point p. Coordinates of gradient vector are given as\+:

$ g_x = \frac{2x}{a^2}, $ $ g_y = \frac{2y}{b^2}, $ $ g_z = \frac{2z}{c^2}. $ ~\newline
