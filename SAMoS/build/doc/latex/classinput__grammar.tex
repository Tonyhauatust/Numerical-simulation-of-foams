\doxysubsection{Detailed Description}
This is a parser for parsing command that contain input file generator seed.

For example\+:

input test.\+dat

This parser will return the file name (\textquotesingle{}test.\+dat\textquotesingle{} in this case)

\begin{DoxyNote}{Note}
We use the same parser to parse internal messages command such as in messages terminal
