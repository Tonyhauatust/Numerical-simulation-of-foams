integrator_grammar Class Reference

Detailed Description

This is a parser for parsing command that contain controls the integrator. Structurally, it is very similar to the potential parser (

See also
parse_potential.hpp) This parser extracts the integrator type and returns it together with the list of parameters that will be parsed separately.

for example:

integrator nvt { dt = 0.005; Tstart = 1.0; Tend = 0.5; tau = 0.5; }

This parser will integrator type (in this case "nvt") and return the rest of the line for post-processing.
