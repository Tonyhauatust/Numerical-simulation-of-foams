population_grammar Class Reference

Detailed Description

This is a parser for parsing command that contain population control. Structurally, it is very similar to the command parsed (

See also
parse_command.hpp) This parser extracts the population line type and returns it together with the list of parameters that will be parsed separately.

for example:

population random { group=all; division_rate=2.0; death_rate = 0.5; freq = 100 }

This parser will extract the population type (in this case "random") and return the rest of the line for post-processing.
