PairVelocityAlign Class Reference

Detailed Description

PairVelocityAlign implements the "velocity" type alignment between neighbouring particles. For all particles within the cutoff distance ( being neighbour list cutoff distance) we compute torque on the particle as (for nematic coupling) and as (for polar case), where is the coupling constant and is velocity of j-th neighbour.
