Potential Class Reference

Detailed Description

Potential class handles all potentials (pair interactions, external forces, bonds and angles) present in the system. All potentials are stored in two STL maps which both have strings as keys while items are pointers (std shared_ptr) to the PairPotential and ExternalPotential abstract classes, respectively. The integrator part of the code calls member functions of this class to perform actual force and potential computations.
