IntegratorNematic Class Reference

Detailed Description

IntegratorNematic class handles dynamics the active nematic (spatial part) as defined in Eq. (2) of E. Bertin, et al. arXiv:1305.0772 Alignment part follows the standard XY-model-like dynamics
