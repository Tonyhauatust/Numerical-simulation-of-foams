external_grammar Class Reference

Detailed Description

This is a parser for parsing command that contain external potentials. Structurally, it is very similar to the pair potential parser (

See also
parse_potential.hpp) This parser extracts the potential line type and returns it together with the list of parameters that will be parsed separately.

for example:

pair_potential gravity { g = 9.81 }

This parser will extract the potential type (in this case "gravity") and return the rest of the line for post-processing.
