Particle Class Reference

Detailed Description

Particle class This is a basic unit in the simulation. It contains relevant parameters that describe each active particle, such as its position, velocity, type, etc.

Note
particle coordinates are given in the external, 3d Euclidean space. Its confinement to a given manifold (curved space) is enforced via constraints.
