IntegratorLangevin Class Reference

Detailed Description

IntegratorLangevin class implements Langevin dynamics for the particle position. Particle director will not be integrated.

void integrate_baoab ( )
private

Integrate using BAOAB method (defualt)

Integrates stochastic equations of motion using Langevin dynamics. The implementation is based on the BAOAB algorithm in the book "Molecular Dynamics: With Deterministic and Stochastic Numerical Methods" by Ben Leimkuhler and Charles Matthews Interdisciplinary Applied Mathematics, Vol. 39, Springer (page 271)

Note
This integrator applies only to the particle position and does not implement activity. In order to use activity, you should define, e.g. external self propulsion.
