Construct the system.

System constructor The constructor reads in coordinates from a file. If there is an error, an exception will be thrown.

File format:

  1. Any line starting with # sign is a comment and is ignored

Otherwise, information about each particle is stored in a one

  1. Particle ids should be stored in the incremental order
  2. Particle line [id] [type] [radius] [x] [y] [z] [vx] [vy] [vz] [omega]
Parameters
input_filenamename of coordinate file
msgPointer to the Messenger object
boxPointer to the simulation box object
