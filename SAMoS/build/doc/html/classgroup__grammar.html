group_grammar Class Reference

Detailed Description

This is a parser for parsing command that contain group directives.

For example:

group small { type = 1 } group R2 { radius = 2.0 }

This parser will extract group name name (in this case "small" and "R2") and return the rest of the line for post-processing.
