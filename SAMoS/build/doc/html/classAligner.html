Aligner Class Reference

Detailed Description

Aligner class handles all director aligners (pairwise and external field type) present in the system. All aligners are stored in two STL maps which both have strings as keys while items are pointers (boost shared_ptr) to the PairAlign and ExternalAlign abstract classes, respectively. The integrator part of the code calls member functions of this class to perform actual alignment computation.
