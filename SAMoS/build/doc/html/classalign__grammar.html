align_grammar Class Reference

Detailed Description

This is a parser for parsing command that contain pairwise alignment. Structurally, it is very similar to the command parsed (

See also
parse_command.hpp) This parser extracts the pairwise aligner line type and returns it together with the list of parameters that will be parsed separately.

for example:

pair_align mf { J = 1.0; }

This parser will extract the pairwise aligner type (in this case "mf") and return the rest of the line for post-processing.
