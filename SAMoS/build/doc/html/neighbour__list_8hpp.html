Dot product between two vectors given by three particles.

Auxiliary function for computing dot product between two vectors defined by three points. This is used to to determine if a particle needs to be mirrored when building the intial triangulation.

Mirror image of a point with respect to an edge.

Auxiliary function for finding a mirror image of a point relative to the edge defined by two other points This is used to to determine the position of the mirrored particle that sits opposite to a boundary edge and has its associated angle being obtuse.
