potential_grammar Class Reference

Detailed Description

This is a parser for parsing command that contain pair potentials. Structurally, it is very similar to the command parsed (

See also
parse_command.hpp) This parser extracts the potential line type and returns it together with the list of parameters that will be parsed separately.

for example:

pair_potential lj { eps = 1.0; sigma = 1.0; r_cut = 2.5; }

This parser will extract the potential type (in this case "lj") and return the rest of the line for post-processing.
